A suite of useful JavaScript plugins and functions for front-end development. These plugins are to help solve the most common challenges associated with a build and below are some examples of it in action. To see the documentation on how these can be configured view the readme file.

Lazy Load

Open up your console to see the loading of these images 100px before they ennter the screen. Look at your page to see the highlighting of the copy in this section with the hex value of #ccc. Side-note: if your scrolling fast on a slow network, may not help, but if slow/normal scrolling on a normal network, you shouldn't see a loading blip.


It's nearly inevitable your website will need some JS plugin's and functions for it to work. These are made to work with Cash (with jQuery still an option) as the only dependency.

Within are some common plugin's to help enhance your website.

Some nice features are their is some shared syntax in the way they all operate.

For Example: all have options that can be plugged in as a data attribute, in JSON format (loosely written somehat)
